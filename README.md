🛡️ Disable Windows Defender & Security Services

Простая и эффективная утилита для отключения встроенной антивирусной защиты Windows через редактирование системного реестра.

---

⚠️ ВАЖНОЕ ПРЕДУПРЕЖДЕНИЕ

Данная программа отключает:

· Microsoft Defender Antivirus
· Службы Real-Time Protection
· Облачную защиту Spynet
· SmartScreen
· Отправку образцов в Microsoft

Это делает систему уязвимой!

Используйте утилиту только в лабораторных средах или на изолированных системах. Автор не несёт ответственности за любой ущерб, нанесённый использованием данной программы. Запуск программы подразумевает ваше полное согласие с этими условиями.

---

📋 Системные требования

· Windows 10 / Windows 11
· Права администратора (обязательно!)

---

🚀 Установка и запуск

Способ 1: Скачать готовый .exe

1. Перейдите в раздел Releases
2. Скачайте последнюю версию DisableMsMpEng.exe
3. Запустите от имени администратора (правый клик → «Запуск от имени администратора»)

Способ 2: Скомпилировать из исходников

```bash
gcc -Wall -Werror -static -O2 DisableMsMpEng.c -o DisableMsMpEng.exe
```

Затем запустите с правами администратора.

🧠 Как это работает

Программа последовательно изменяет ключи реестра в разделах:

Раздел Действие
HKLM\SOFTWARE\Policies\Microsoft\Windows Defender Отключает антивирус и службы
HKLM\...\Real-Time Protection Отключает мониторинг в реальном времени
HKLM\...\Spynet Отключает облачную защиту и отправку образцов
HKLM\SYSTEM\CurrentControlSet\Services\WinDefend Отключает службу Защитника Windows
HKLM\SYSTEM\CurrentControlSet\Services\MDCoreSvc Отключает службу защиты от вредоносных программ
HKCU\Software\...\Attachments Отключает маркировку загруженных файлов
HKLM\...\Windows\System Отключает SmartScreen

Программа автоматически определяет нужный корневой раздел (HKLM, HKCU, etc.) на основе переданного пути.

🔧 Архитектура

Код написан на чистом C, использует только WinAPI и стандартные библиотеки.
Основная логика вынесена в функцию vault_createkey_dword, которая:

· Определяет корневой раздел реестра по строке пути
· Отделяет путь от имени ключа
· Записывает DWORD-значение

Все ключи и их значения собраны в компактные массивы, что делает код легко расширяемым:

```c
const char *regedit_params[] = {...};
DWORD value_param[] = {...};
```

Добавление нового ключа занимает одну строку.

---

🛠️ Известные особенности

· При отключении защитника может потребоваться перезагрузка для применения всех изменений.
· Windows может автоматически восстановить настройки после обновлений.
· Утилита не завершает процессы и не удаляет файлы — только изменяет ключи реестра.
